-------------------------------------
------Compilando o instalador--------
-------------------------------------

Existem 2 formas de compilar o instalador: Através da linha de comando ou utilizando o Ant.

Para compilar o instalador através da linha de comando
Abra uma janela DOS;
Vá até o diretório onde está o seu projeto, na raíz. No caso Jogo21;
Certifique-se que o diretório bin do IzPack está no path de execução de seu sistema:
Digite: 

compile instalador.xml -b . -o install.jar -k standard
	ou simplesmente 
compile isntalador.xml

Para incluir a criação do instalador no arquivo build.xml do seu projeto
Defina a anttask como abaixo
<taskdef name="izpack" classpath="${izpack.dir}/lib/compiler.jar"
classname="com.izforge.izpack.ant.IzPackTask"/>
Crie a task como no modelo
<izpack input="extras/installer.xml"
output="dist/install/produto-install.jar"
installerType="standard" basedir="${basedir}"
izPackDir="${izpack.dir}/"/>
A property ${izpack.dir} deve apontar para o diretório de instalação do IzPack.
